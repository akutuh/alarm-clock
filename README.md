# Alarm-clock app

Alarm clock made with react
[Live Demo](https://lit-shelf-09682.herokuapp.com/)

## Run the project localy

**0 -** install node
**1 -** clone the project and install dependencies:

```sh
git clone https://github.com/akutuh/alarm-clock.git
cd alarm-clock
npm install
```

**2 -** start the app

```sh
npm start
```
